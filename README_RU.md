## Особенности форка

Этот форк является инженерной адаптацией оригинального DeepSeek++ для **мобильной разработки на Android** и **автоматизированного деплоя**. Внесены следующие ключевые изменения:

- **Полноценная работа на мобильных браузерах (Kiwi / Lemur):** Обойдено ограничение API боковой панели (`chrome.sidePanel`) — настройки MCP и навыков открываются в отдельной вкладке через опцию `options_ui` в `wxt.config.ts`.
- **Автоматизация деплоя через Termux:** Python-скрипт [`deploy.py`](deploy.py) автоматизирует полный цикл: `git add` → `commit` → `push` → мониторинг GitHub Actions → скачивание готового `.zip` в папку `/sdcard/Download/`.
- **Оптимизация CI/CD:** Из воркфлоу GitHub Actions удалены «качественные ворота» (`i18n quality gates`), которые блокировали кастомные сборки. Исправлены права доступа для автоматического создания релизов.
- **Русская документация:** Полный перевод `README_RU.md` и адаптация инструкций под специфику форка.

> Текущая стабильная версия форка: **v0.7.2+**

## Позиционирование продукта

DeepSeek++ — это открытое расширение для браузера, предназначенное для [DeepSeek](https://chat.deepseek.com) и поддерживающее Chrome, Edge и Firefox. Оно превращает веб-версию DeepSeek в AI Agent рабочую среду, позволяя в едином браузерном рабочем процессе использовать русскоязычный/англоязычный интерфейс, инструменты MCP, долговременную память, навыки (Skills), предустановки системных промптов, веб-поиск, чтение веб-страниц, экспорт диалогов и автоматизацию по расписанию.

Если вы ищете расширение DeepSeek для Chrome, инструменты MCP для DeepSeek, плагин памяти, экспорт диалогов или AI Agent для DeepSeek — DeepSeek++ это локально-ориентированная расширенная среда.

## Ключевые возможности

### Боковая панель диалога
- **Опциональный вход** — после включения в настройках на боковой панели появляется вкладка «Диалог» для отправки сообщений DeepSeek.
- **Отправка выделенного текста** — выделите текст на любой странице и отправьте его в диалог правой кнопкой мыши.
- **Сценарии правой кнопки** — настройте шаблоны промптов для быстрой обработки выделенного текста.
- **API-ключ DeepSeek** — при его настройке расширение работает на любых сайтах; без него — только на странице DeepSeek.
- **Отдельные сессии** — поддержка создания новых сессий, чтобы не пересекаться с текущим диалогом.

### Русский и английский интерфейс
- Выбор языка: вслед за браузером, упрощённый китайский или English.
- Боковая панель, контекстное меню, результаты инструментов, встроенные навыки и подсказки модели используют выбранный язык.
- Пользовательские данные (память, пресеты, навыки, автоматизация) остаются в оригинале.

### Контекст проекта и артефакты
- **Библиотека проектов** — организация репозиториев, веб-страниц, локальных файлов в рамках одного проекта.
- **Выборочное добавление** — добавление в диалог только релевантных материалов.
- **Однофайловые артефакты** — генерация скриптов, Markdown, JSON, HTML.
- **Пакетные артефакты** — многофайловые результаты упаковываются в ZIP.

### Нативные вызовы инструментов
- Автоматическое распознавание и выполнение инструментов без ручного вмешательства.
- Скрытие технических деталей — показывается только краткий результат.
- Поддержка нескольких последовательных вызовов в одном ответе.
- Восстановление записей выполненных инструментов после обновления страницы.

### Экспорт диалогов
- Экспорт текущего диалога прямо из панели инструментов ответа DeepSeek.
- Форматы: HTML, Markdown, PDF.
- Режим «только для чтения» — скрываются внутренние подсказки расширения и метки вызовов инструментов.
- Экспорт отдельных сообщений в Markdown.

### Встроенные веб-инструменты
- **Веб-поиск** — модель может искать информацию в реальном времени.
- **Получение веб-страницы** — модель может прочитать и проанализировать содержимое указанной страницы.
- Управление инструментами на боковой панели («Инструменты»).
- Автоматическое продолжение: после завершения поиска модель продолжает генерацию.

### Агентное выполнение задач
- Пошаговое продолжение: модель может последовательно вызывать инструменты, пока задача не будет решена.
- Отображение шагов в свёрнутых блоках.
- Возможность ручной остановки длительных задач.
- Восстановление состояния после обновления страницы.

### MCP-система
- Подключение удалённых или локальных MCP-сервисов (HTTP, SSE, stdio bridge).
- Автоматическое выполнение инструментов (можно переключить в ручной режим).
- Управление правами и тестирование подключений на боковой панели.
- Результаты инструментов автоматически возвращаются в ту же сессию.

### OfficeCLI
- Встроенный навык `/officecli` для работы с `.docx`, `.xlsx`, `.pptx`.
- Официальная библиотека навыков от OfficeCLI (DOCX, XLSX, PPTX, финансовые модели, питч-деки и др.).
- Работа через Shell MCP.
- Автоматическая установка командной версии OfficeCLI через `deepseek-pp-shell-host`.

### Система памяти
- **Автоматическое запоминание** — модель сохраняет ключевую информацию в процессе диалога.
- **Умная инъекция** — при старте новой сессии автоматически подставляется релевантный контекст.
- Четыре типа: профиль пользователя (`user`), фидбек (`feedback`), тема диалога (`topic`), справочные материалы (`reference`).
- Управление на боковой панели: просмотр, редактирование, закрепление, удаление.

### Система навыков (Skills) и слэш-команды
- Встроенные навыки, пользовательские и импортированные из GitHub.
- Вызов через `/` в поле ввода.
- GitHub-импорт с отображением источника, версии и лицензии.

### Предустановки системных промптов
- Создание нескольких предустановок глобальных ролей или поведенческих инструкций.
- Активация одной предустановки одновременно.
- Автоматическая вставка в начало каждой новой сессии.

### Автоматизация задач
- Запуск вручную или по расписанию (cron / RRULE).
- Каждая автоматическая задача работает в отдельной сессии.
- Минимальный интервал планировщика: 15 минут.
- Управление задачами на боковой панели: пауза, редактирование, удаление.

## Установка

### Установка на Android (Kiwi / Lemur) с автоматическим деплоем

Этот форк оптимизирован для мобильной разработки. Для автоматической сборки и установки выполните:

1. Установите [Termux](https://termux.com/) на Android.
2. В Termux выполните:
   ```bash
   pkg update && pkg install git python nodejs-lts
   git clone https://github.com/LeonidYasin/deepseek-pp.git
   cd deepseek-pp
   npm install
   ```
3. Запустите автоматический деплой:
   ```bash
   python deploy.py
   ```
   Скрипт выполнит коммит, пуш, отследит сборку в GitHub Actions и скачает готовое расширение `.zip` в `/sdcard/Download/`.
4. В браузере Kiwi или Lemur перейдите в `chrome://extensions`, включите «Режим разработчика» и нажмите «+ (из .zip)» или «Загрузить распакованное».

### Установка на десктоп (Chrome / Edge / Firefox) из исходников

```bash
git clone https://github.com/LeonidYasin/deepseek-pp.git
cd deepseek-pp
npm install
npm run build      # Соберет Chrome MV3
npm run build:all  # Соберет Chrome, Edge и Firefox
```

После сборки загрузите расширение в браузере через режим разработчика:
- Chrome: `chrome://extensions/`
- Edge: `edge://extensions/`
- Firefox: `about:debugging#/runtime/this-firefox`

### Сборка для Android WebView (для разработчиков мобильных интеграций)

```bash
npm run build:android
```
