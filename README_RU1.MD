#### Позиционирование продукта
DeepSeek++ расширяет стандартный веб-интерфейс DeepSeek, добавляя инструменты, которых нет «из коробки»: поддержку протокола MCP, долговременную память, систему навыков (Skills), автоматизацию по расписанию и продвинутый экспорт. Проект ориентирован на разработчиков и опытных пользователей, стремящихся выжать максимум из взаимодействия с ИИ прямо в браузере.

### 🌟 Ключевые возможности нашего форка (v0.7.2+)

В данной версии была проведена глубокая адаптация расширения для работы в мобильных окружениях и автоматизации процессов разработки:

1.  **Поддержка мобильных браузеров (Kiwi/Lemur):**
    *   Исправлено ограничение, мешавшее открытию настроек на Android. В манифест добавлена поддержка `options_ui`, что позволяет открывать панель управления MCP и навыками в полноэкранном режиме, обходя отсутствие API боковой панели в мобильных Chromium-браузерах.
2.  **Скрипт автоматического деплоя (`deploy.py`):**
    *   Разработана утилита на Python для Termux/Linux, которая полностью автоматизирует цикл: коммит -> создание тега -> пуш на GitHub -> ожидание облачной сборки -> скачивание готового `.zip` расширения прямо в папку загрузок телефона.
3.  **Оптимизация CI/CD пайплайна:**
    *   Из GitHub Actions удалены избыточные и жесткие проверки локализации (i18n quality gates), которые блокировали кастомные конфигурации сборки.
    *   Настроены автоматические права записи для `GITHUB_TOKEN`, что позволило облачному сборщику самостоятельно публиковать релизы.

### 🛠 Основные функции (Оригинальные)

##### Поддержка Model Context Protocol (MCP)
Плагин позволяет DeepSeek общаться с внешними MCP-серверами через HTTP, SSE или stdio bridge. Это дает модели возможность вызывать локальные инструменты, читать файлы и выполнять команды, превращая браузер в среду выполнения задач.

##### Система памяти (Memory)
*   **Автоматическое запоминание:** ИИ распознает и сохраняет ключевую информацию в процессе диалога.
*   **Умная инъекция:** При старте нового чата расширение автоматически встраивает релевантные фрагменты памяти на основе контекста.
*   **Типы памяти:** Поддержка профилей пользователя, фидбека, топиков и справочных материалов.

##### Система навыков (Skill) и слэш-команды
*   Возможность создания кастомных системных промптов и сложных цепочек действий.
*   Вызов навыков через ввод `/` в поле чата для мгновенного внедрения нужных инструкций.
*   Импорт сторонних навыков напрямую из репозиториев GitHub.

##### Автоматизация задач (Automation)
*   **Планировщик:** Запуск задач вручную или по расписанию (cron/RRULE).
*   **Изолированные сессии:** Каждая автоматическая задача может работать в собственном диалоге для отслеживания конкретной темы.

##### Продвинутый экспорт
Кнопки сохранения встроены прямо в тулбар ответов DeepSeek. Поддерживается экспорт в Markdown, HTML и PDF со всеми метаданными и ссылками на артефакты.

### 📦 Установка

**Для мобильных устройств (Android):**
1.  Установите **Kiwi Browser**.
2.  Скачайте последний релиз нашего форка из раздела [Releases](https://github.com/LeonidYasin/deepseek-pp/releases) (файл `deepseek-mobile.zip`).
3.  В Kiwi перейдите в `chrome://extensions`, включите "Режим разработчика" и загрузите архив через кнопку "+ (из .zip)".

**Для разработчиков (Сборка из исходников):**
```bash
# Установка зависимостей
npm install

# Сборка под Chrome/Edge
npm run build:chrome

# Сборка под Firefox
npm run build:firefox
```
*Для автоматической сборки и загрузки на телефон используйте:* `python deploy.py`.
