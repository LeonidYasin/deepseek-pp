## DeepSeek++ 0.4.3

0.4.3 是长任务稳定性和互动反馈增强版本，重点改善 DeepSeek 校验兼容、Agent 式持续执行节奏和悬浮宠物状态反馈。

### 主要变化

- 更新本地 DeepSeek 校验计算方式，减少长任务、自动化和工具续跑过程中因平台校验失败而中断的情况。
- Agent 式持续执行会在连续请求之间自动留出间隔，降低多步工具续跑过快触发失败的概率。
- 空续跑会显式失败并保留已有步骤状态，避免长任务看似完成但没有实际输出。
- DeepSeek 小鲸鱼新增状态台词气泡，会在思考、输出、工具执行和空闲状态展示并轮播反馈。
- 新增标准 issue 表单和模板检查，未填写必要信息的问题会自动关闭并提示补充。
- README 新增 0.4.3 变更回顾，并继续保留 0.4.2 / 0.4.1 / 0.4.0 / 0.3.0 / 0.2.0 的折叠历史。
